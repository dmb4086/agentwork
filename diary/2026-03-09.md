### Research Findings

**Postfix/Dovecot:**
- Can create mailboxes programmatically via SQL
- Virtual users (no system accounts needed)
- BUT: DNS complexity, deliverability challenges

**AWS SES:**
- Great for sending
- Receiving requires WorkMail (human console access required)
- Not fully agent-native

**Hybrid (Recommended):**
```
Receiving: Mailgun/Postmark → Webhook → API
Sending: AWS SES
Storage: PostgreSQL + S3
IMAP: Custom lightweight server
```
- Best deliverability
- Fully programmatic
- Scalable
